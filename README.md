# Matt King :seedling:

A simple web page about me :upside_down_face:

Check it out [here](https://mattking.herokuapp.com/).

## Install and run

Clone the repository
```
git clone https://github.com/matthewdking/mattking.git
```
Install modules
```
npm install
```
Start
```
npm start
```
